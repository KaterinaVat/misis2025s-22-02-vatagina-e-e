# Отчет по лабораторной работе №4

## Цель работы
Разработка системы генерации тестовых изображений с эллиптическими объектами, их детектирования и оценки качества работы детектора.

## Выполненные задачи

### 1. Генератор тестовых данных (task04-01)
Реализовано консольное приложение для генерации:
- Тестовых изображений размером N×M клеток по 256×256 пикселей
- Клетки с равномерным фоном
- Светлых эллипсов со случайными параметрами:
  - Положение (с отступом 32 пикселя от границ клетки)
  - Яркость
  - Размер
  - Ориентация
  - Степень размытия
- Добавлен равномерный шум

**Параметры командной строки:**
```
.\task0401.exe <config_path> <image_path> [--seed VALUE]
```
![Коллаж результатов работы](https://github.com/KaterinaVat/misis2025s-22-02-vatagina-e-e/raw/main/prj.lab/lab04/output/collage/collage.png)

**Функциональность:**
- При отсутствии других параметров генерирует файл конфигурации с параметрами по умолчанию
- Поддержка seed для воспроизводимости результатов
- Сохранение эталонной разметки (координаты и параметры эллипсов)

### 2. Детектор объектов (task04-02)
Реализован алгоритм детектирования эллипсов:
1. Бинаризация изображения (адаптивный порог)
2. Морфологическая обработка (закрытие для устранения шума)
3. Анализ компонент связности
4. Аппроксимация эллипсами найденных областей

**Параметры командной строки:**
```
task04-02 <image_path> <detection_result_path>
```
